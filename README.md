# 🔐 CryptoDetective – Educational Cryptography Investigation Game

CryptoDetective is an interactive browser-based educational game that teaches message integrity verification using hashing, digital signatures, and encryption analysis through investigation-style puzzles.

## 🌐 Live Demo

Play the game online:

[https://yourusername.github.io/cryptodetective/](https://dnialhkm.github.io/CryptoProject/)

---

## 🎯 Project Objective

This project helps students understand how cryptographic techniques verify authenticity and detect tampering in digital communication.

Players learn to:

* Compare message hashes
* Verify digital signatures
* Understand encryption purpose
* Detect modified communications
* Analyze digital evidence step-by-step

---

## 🕹️ Game Features

* 3 interactive investigation cases
* Step-by-step cryptographic verification workflow
* AI hint assistant support
* Evidence-based decision system
* Final detective completion screen
